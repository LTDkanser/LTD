Game Engines

Базовое программное обеспечение для разработки и запуска видеоигр, с которыми мы работаем

Unreal Engine

Мощный движок от Epic Games для AAA-игр и реалистичной 3D-графики.

  • Nanite — геометрия нового поколения
  • Lumen — глобальное освещение
  • Blueprint — визуальное программирование
  • Cross-platform (PC, Console, Mobile)

CryEngine

Движок от Crytek с передовой физикой и графикой.

  • Продвинутая физика разрушений
  • Реалистичная водная симуляция
  • Sandbox — открытый редактор
  • Реалистичная графика

Unity

Универсальный движок для мобильных, инди и кроссплатформенных игр.

  • C# — основной язык
  • Asset Store — marketplace
  • 2D и 3D разработка
  • Мобилы + WebGL + AR/VR
